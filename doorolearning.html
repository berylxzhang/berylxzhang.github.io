Dooro Learning - IOS App

  • Category: App design
  • Client: Bear Bear Company (self-founded)
  • Project date: 01 July, 2021
  • Project URL: https://github.com/berylxzhang/Dooro-Learning

portfolio features

1. Sign up a new account with email address and self-defined password
2. Retreive an account when password is forgotten by sending a password reset link to the email address and check whether the input email is still valid
3. Log in using email and password which have been registered, and validate whether the account is existing and whether the password matches the email
4. Log in and log out using facebook account
5. Log in and log out using Google account
6. Flashcard Mode has add, delete, search, guess, and explanation functionality
7. Manage each user's login information using firebase and corresponding database using core data
8. Hangman Mode allow users to play with the words they have already added
9. The app can work perfectly on multiple IOS devices by setting the constraints reasonably
